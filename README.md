# AudioFill
Проект создания визуализации для платформы Android TV с использованием библиотеки Flutter.

## Особенности реализации
Работа без использования арзитектурных паттернов через Provider.
Считывание уровня окружающего звука происходит дискретно.
Проект является демонстратором использования микрофона для визуализации звукового окружения на телевизоре с ОС Android TV.
